# 개발 로그

젠슨황 KR 트래커 작업 이력입니다.  
날짜별 파일로 관리하며, 주요 기능 변경·코드 설명·배포 이력을 기록합니다.

## 파일 구조

```
docs/logs/
├── README.md          ← 이 파일
├── 2026-06-05.md      ← 날짜별 작업 로그
├── 2026-06-06.md
└── ...
```

## 작성 규칙

- 파일명: `YYYY-MM-DD.md`
- 섹션: 작업 요약 / 주요 변경 (코드 설명) / 배포 / 다음 예정
- 코드 설명은 비개발자도 이해할 수 있는 수준으로 작성
